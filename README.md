This repository provides datasets for the study of contradiction detection between Brazilian legislative bills.

---

## 1. `bills.csv`

Dataset containing legislative bills collected from the Legislative Assembly of Minas Gerais (ALMG).

Each row represents a single legislative bill.

**Columns:**

- `tipo`: Type of the bill  
- `numero`: Bill number  
- `ano`: Year of the bill  
- `complemento`: Additional metadata or descriptor (if available)  
- `texto_original`: Original bill text  
- `texto_pre_processado`: Preprocessed version of the bill text  

---

## 2. `similar_pairs.csv`

Dataset of semantically similar bill pairs identified through institutional procedures.

Each row represents a pair of related legislative bills.

**Columns:**

- `tipo_1`: Type of the first bill  
- `numero_1`: Number of the first bill  
- `ano_1`: Year of the first bill  
- `complemento_1`: Additional metadata for the first bill  
- `texto_pre_processado_1`: Preprocessed text of the first bill  

- `tipo_2`: Type of the second bill  
- `numero_2`: Number of the second bill  
- `ano_2`: Year of the second bill  
- `complemento_2`: Additional metadata for the second bill  
- `texto_pre_processado_2`: Preprocessed text of the second bill  

---

## 3. `contradictory_real_pairs.csv`

Dataset of real contradictory bill pairs manually curated by domain specialists based on legislative analysis documents.

Each row represents a pair of bills identified as contradictory.

**Columns:**

- `tipo_1`: Type of the first bill  
- `numero_1`: Number of the first bill  
- `ano_1`: Year of the first bill  
- `complemento_1`: Additional metadata for the first bill  
- `texto_pre_processado_1`: Preprocessed text of the first bill  

- `tipo_2`: Type of the second bill  
- `numero_2`: Number of the second bill  
- `ano_2`: Year of the second bill  
- `complemento_2`: Additional metadata for the second bill  
- `texto_pre_processado_2`: Preprocessed text of the second bill  

- `tema`: Thematic category of the contradiction (e.g., rodeo, gender, vaccination)

---

## 4. `contradictory_synthetic_pairs.csv`

Dataset of synthetic contradictory legislative bills generated using a Large Language Model (LLM).

Each example consists of a real legislative bill and a synthetically generated contradictory version.

**Columns:**

- `tipo`: Type of the bill  
- `numero`: Bill number  
- `ano`: Year of the bill  
- `complemento`: Additional metadata or descriptor (if available)  
- `texto_real`: Original legislative bill text  
- `texto_sintetico_gerado`: Synthetic contradictory bill generated by the LLM  

---

## 5. `finetuning_dataset.csv`

Dataset used for supervised fine-tuning experiments.

Each row represents a pair of texts labeled as contradictory or non-contradictory.

**Columns:**

- `label`: Classification label (`1` for contradictory, `0` for non-contradictory)  
- `txt_1`: Text of the first bill  
- `txt_2`: Text of the second bill  
- `fonte`: Source/type of the pair (e.g., synthetic, similar, random)

---

## 6. `few_shot_{i}.csv`

Few-shot examples used in each experimental fold.

Each file contains the demonstration examples provided to the LLM during few-shot prompting.

**Columns:**

- `label`: Classification label (`1` for contradictory, `0` for non-contradictory)  
- `txt_1`: Text of the first bill  
- `txt_2`: Text of the second bill  
- `fonte`: Source/type of the pair  
- `tema`: Thematic category of the pair  

---

## 7. `test_{i}.csv`

Test datasets used in each experimental fold.

Each file contains the evaluation pairs used to assess model performance.

**Columns:**

- `label`: Classification label (`1` for contradictory, `0` for non-contradictory)  
- `txt_1`: Text of the first bill  
- `txt_2`: Text of the second bill  
- `fonte`: Source/type of the pair  
- `tema`: Thematic category of the pair
